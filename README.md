# UTMStarter

**UTMStarter** — десктопное приложение для Windows, которое упрощает управление несколькими экземплярами УТМ ЕГАИС на одном компьютере.

---

## Что умеет приложение

- **Запуск и остановка УТМ** — управление несколькими УТМ одновременно, каждый на своём порту с отдельным токеном
- **Мониторинг очередей** — отображение входящих и исходящих документов в режиме реального времени
- **Работа с ТТН** — просмотр непринятых накладных, отправка актов приёмки/отказа, перезапрос документов
- **Автоматизация** — контроль наличия необработанных ТТН при старте модуля, автоматическая приёмка ТТН
- **Управление сертификатами** — просмотр ГОСТ-сертификатов с токенов Rutoken, отслеживание сроков истечения
- **Обновления** — автоматическая проверка и установка новых версий

---

## Системные требования

| | |
|---|---|
| **ОС** | Windows 10 / Windows 11 (64-bit) |
| **Права** | Администратор (для управления службами) |
| **УТМ** | Установленный УТМ ЕГАИС (`C:\UTM`) |
| **Токен** | Rutoken ECP с действующим ГОСТ-сертификатом |
| **Интернет** | Для обмена документами с серверами |

## Установка

### 1. Скачайте установщик

Перейдите в раздел [**Releases**](../../releases) и скачайте файл `UTMStarter-Setup-x.x.x.exe` из последнего релиза.

### 2. Запустите установщик

Дважды кликните по скачанному файлу. Если Windows предупреждает о неизвестном издателе — нажмите **«Подробнее»** → **«Выполнить в любом случае»**.

> Установщик распакует файлы приложения и создаст ярлык на рабочем столе.

### 3. Запустите приложение

Установщик создаст ярлык для запуска приложения с необходимыми правами. Но если что-то пошло не так, запускайте UTMStarter **от имени администратора** — это необходимо для управления службами Windows.

> Правой кнопкой по ярлыку → **«Запуск от имени администратора»**

### 4. Войдите в аккаунт

После запуска введите email и пароль. Если у вас нет аккаунта — нажмите **«Создать аккаунт»** прямо в окне входа.

## Обновление

Приложение проверяет наличие обновлений **автоматически при каждом запуске**. Если доступна новая версия — приложение будет обновлено, дополнительных действий не требуется.

Также можно скачать обновление вручную из раздела [**Releases**](../../releases) и распаковать в папку приложения с заменой файлов.

## Часто задаваемые вопросы

**УТМ не запускается / служба не устанавливается**

Убедитесь, что приложение запущено от имени администратора и что УТМ установлен по стандартному пути `C:\UTM`.

**Токен не определяется**

Проверьте, что токен вставлен и доступен в Панели управления Рутокен. 
Извлеките токен и вставьте заново.
Нажмите «Обновить» на странице «Модули».

**Приложение просит повторно войти**

Срок действия сессии истёк. Введите пароль в появившемся окне — активная работа не прервётся.

## 🔒 Персональные данные
 
UTMStarter взаимодействует с серверной частью сервиса для обеспечения работы приложения. В процессе работы на сервер могут передаваться:
 
- адрес электронной почты (используется как логин)
- версия приложения
- метаданные ГОСТ-сертификатов с ключевых носителей (серийный номер, Common Name, срок действия) — **без закрытых ключей**
 
Мы не передаём эти данные третьим лицам и не используем их в коммерческих целях. Данные хранятся на сервере в объёме, необходимом для функционирования сервиса.
 
Продолжая использовать приложение, вы соглашаетесь с указанными условиями обработки данных. Если вы не согласны — пожалуйста, не используйте приложение.
